# Apriori Algorithm for Market Basket Analysis

## Overview

This project implements the **Apriori Algorithm** to discover association rules from a market basket dataset. The algorithm identifies frequently purchased item combinations using **Support**, **Confidence**, and **Lift**, helping businesses understand customer purchasing behavior.

## Dataset

* **Dataset:** Market Basket Optimisation
* **Transactions:** 7,501
* **Maximum Items per Transaction:** 20

## Technologies Used

* Python
* Pandas
* Apyori
* Matplotlib

## Project Workflow

1. Load the dataset.
2. Convert transactions into a list format.
3. Apply the Apriori algorithm.
4. Generate association rules.
5. Evaluate rules using:

   * Support
   * Confidence
   * Lift
6. Visualize the results.

## Visualization

The project includes a visualization of the discovered association rules to help analyze purchasing patterns.

## Results

The Apriori algorithm successfully identified frequent itemsets and generated association rules based on the specified minimum support, confidence, and lift thresholds.

## Applications

* Market Basket Analysis
* Product Recommendation
* Cross-selling
* Store Layout Optimization
* Inventory Planning
