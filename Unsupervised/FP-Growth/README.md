# FP-Growth Algorithm

This project demonstrates the **FP-Growth** algorithm for frequent pattern mining using a market basket dataset.

## Libraries
- Pandas
- Matplotlib
- mlxtend

## Workflow
- Load dataset
- One-hot encode transactions
- Apply FP-Growth
- Generate association rules
- Visualize frequent itemsets and rules

## Visualizations
- Top 10 Frequent Itemsets
- Support Distribution
- Confidence vs Lift Scatter Plot

## Conclusion
FP-Growth efficiently mines frequent itemsets without candidate generation, making it faster than Apriori on large datasets.
